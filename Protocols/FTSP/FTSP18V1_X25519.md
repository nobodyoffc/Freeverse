## Abstract

**`X25519`** (FC-JDK) is **Curve25519** scalar–mult agreement: **32-byte** private key, **32-byte** public key, **32-byte** shared secret via **`X25519Agreement`** (BouncyCastle). Optional key derivation **`sharedSecretToSymkey` / `asyKeyToSymkey`** uses [FTSP13](FTSP13V1_HKDF.md) **`HKDF.hkdf`** with **`info = ASCII "hkdf"`** and caller-supplied **salt** (typically the **12-byte** AES-GCM IV in [FTSP19](FTSP19V1_X25519AesGcm256.md)). **`generatePublicKey`** derives the **32-byte** public key from a **32-byte** private key.

## Specification

|Item|Bytes|
|---|---|
|Private key|32|
|Public key|32|
|Shared secret|32|

### Procedures

- **`getSharedSecret(priKey32, pubKey32)`**: validates lengths; **`X25519PrivateKeyParameters` / `X25519PublicKeyParameters`**; **`calculateAgreement`** into output buffer.
- **`sharedSecretToSymkey(sharedSecret, nonce)`**: **`HKDF.hkdf(sharedSecret, salt = nonce, info = "hkdf", L = 32)`**.
- **`asyKeyToSymkey`**: shared secret → HKDF; **zeroize** shared secret buffer.
- **`generatePublicKey(priKey32)`**: **`X25519PrivateKeyParameters.generatePublicKey().encode`**.

## Test Vectors

### TV-FTSP18-1 — reciprocal agreement + HKDF

|Item|Value|
|---|---|
|**`priA`**|32 × **`0x41`**|
|**`priB`**|32 × **`0x42`**|
|**`pubA`**|`X25519.generatePublicKey(priA)`|
|**`pubB`**|`X25519.generatePublicKey(priB)`|
|**Property**|`getSharedSecret(priA, pubB)` **equals** `getSharedSecret(priB, pubA)`|

**HKDF:** `salt` = 12-byte hex `2122232425262728292a2b2c`, `info = ASCII "hkdf"`, `L = 32`, **`ikm`** = that shared secret → **`OKM`** (hex) **`7596a67ebf2278a6430ea318b4159d2f73d1720644d01bc5b56185b4ee916899`**.

## Developer JSON example

X25519 material is **not** the secp256k1 FIDs in [FTSP0 §2.1](FTSP0V1_FTSP.md#21-shared-example-keys-developer-json-samples); it uses two **raw 32-byte** private scalars (**`priA`** = 32 × `0x41`, **`priB`** = 32 × `0x42`). **`symkey32`** = `X25519.sharedSecretToSymkey(sharedSecret, saltIv12)` with **`saltIv12`** = 12-byte `000102030405060708090a0b` and HKDF **`info`** = ASCII **`hkdf`**.
